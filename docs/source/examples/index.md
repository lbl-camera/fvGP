The notebooks below walk through common fvGP use cases from basic single-task regression
to large-scale sparse GPs and non-Euclidean input spaces.
Each notebook can be downloaded and run locally after `pip install fvgp`.

| Notebook | Description |
|---|---|
| [Single-task GP](SingleTaskTest.ipynb) | Basic GP regression with the `GP` class |
| [Multi-task GP](MultiTaskTest.ipynb) | Multi-output regression with the `fvGP` class |
| [gp2Scale](gp2ScaleTest.ipynb) | Large-scale sparse GP with distributed computation |
| [Non-Euclidean inputs](NonEuclideanInputSpaces.ipynb) | GPs over strings, graphs, and other non-Euclidean spaces |
